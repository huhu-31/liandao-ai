## 🔧 DNS 配置（绑定到 Vercel）

### 方案 A：使用 Vercel Nameserver（推荐）

#### 1. 获取 Vercel Nameserver
1. 登录 Vercel
2. 进入项目 → Settings → Domains
3. 添加您的域名 `liandao.ai`
4. Vercel 会显示 Nameserver 地址，格式如：
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ```

#### 2. 在域名注册商处修改 Nameserver
1. 登录域名注册商后台（如 NameSilo）
2. 找到域名管理 → DNS 管理
3. 修改 Nameserver 为 Vercel 提供的地址
4. 保存设置

#### 3. 等待生效
- DNS 传播时间：通常 10 分钟 - 24 小时
- 可使用 https://whatsmydns.net 检查全球 DNS 状态

---

### 方案 B：手动配置 DNS 记录

#### 1. 添加 A 记录（根域名）
```
类型：A
主机：@
值：76.76.21.21
TTL：自动（或 3600）
```

#### 2. 添加 CNAME 记录（www 子域名）
```
类型：CNAME
主机：www
值：cname.vercel-dns.com
TTL：自动（或 3600）
```

#### 3. 在 Vercel 验证
1. 回到 Vercel → Settings → Domains
2. 点击 "Verify Configuration"
3. 验证通过后显示 "Valid Configuration"

## ⚠️ 注意事项

### 购买前
- ✅ 确认域名未被注册
- ✅ 检查域名历史（避免被惩罚过的域名）

- ⚠️ 不要频繁修改 DNS 记录

## 🆘 常见问题

### Q: 域名被注册了怎么办？
A: 尝试以下变体：
- 加连字符：`lian-dao.ai`
- 加前后缀：`getliandao.ai`, `liandaoapp.ai`
- 换后缀：`.io`, `.tech`, `.co`
- 用缩写：`ld.ai`（如果可用）

### Q: 中文域名好吗？
A: 不推荐。原因：
- 兼容性差（部分系统不支持）
- 输入麻烦（需要切换输入法）
- 不利于国际化
- 价格偏高

### Q: 需要备案吗？
A: 
- 使用国内服务器/CDN：**必须备案**
- 使用 Vercel/Netlify 等海外服务：**无需备案**
- 建议：初期用海外服务，后期业务稳定后再考虑备案

### Q: 域名多久能生效？
A: 
- 最快：10-30 分钟
- 通常：1-2 小时
- 最慢：24-48 小时（罕见）
