- **方式 A**：点击该区域，在文件选择器中选择 `liandao-ai` 文件夹中的所有文件

### 步骤 3：填写提交信息
在 "Commit summary" 框中输入：

### 步骤 5：等待上传完成
文件将开始上传（约 10-30 秒）

## 🔗 绑定自定义域名（可选）

如果您购买了 `liandao.ai` 域名：

### 步骤 1：在 GitHub 添加 CNAME
1. 进入仓库：https://github.com/huhu-31/liandao-ai
2. 点击 "Add file" → "Create new file"
3. 文件名输入：`CNAME`（注意大写）
4. 内容输入：`liandao.ai`
5. Commit 保存

### 步骤 2：配置 DNS
在域名注册商处添加：
```
类型：A
主机：@
值：185.199.108.153

类型：A
主机：@  
值：185.199.109.153

类型：A
主机：@
值：185.199.110.153

类型：A
主机：@
值：185.199.111.153

类型：CNAME
主机：www
值：huhu-31.github.io
```

### 步骤 3：在 GitHub Pages 设置域名
1. 进入 https://github.com/huhu-31/liandao-ai/settings/pages
2. Custom domain 输入：`liandao.ai`
3. 点击 Save
4. 勾选 "Enforce HTTPS"（DNS 生效后）

等待 DNS 生效（10 分钟 -2 小时），即可通过自定义域名访问！
