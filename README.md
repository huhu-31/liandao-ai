# 极智 AI 员工 - 官方网站


这是一个完整可交互的极智 AI 功能展示网站。


## 功能特性


- ✅ 响应式设计，支持所有设备
- ✅ 平滑滚动动画效果
- ✅ 交互式功能详情展示
- ✅ 应用场景切换
- ✅ 联系表单模态框
- ✅ 现代化渐变设计
- ✅ 完整的功能介绍页面


## 核心功能模块


1. **微信自动机器人** - 无缝接管微信对话，智能客服
2. **智能创作** - 自动剪辑、文案生成
3. **聚合发布** - 一键同步多平台发布
4. **朋友圈精细化运营** - 私域流量管理
5. **远程手机操控** - 语音控制电脑办公
6. **千人千面群发** - 个性化内容推送
7. **自动获客** - 智能识别转化潜在客户
8. **自动工作流** - 全自动营销闭环
9. **智能客服** - 7*24 小时自动回复


## 本地运行


### 方法一：使用 Python 内置服务器
```bash
cd liandao-ai
python -m http.server 8080
```
然后在浏览器访问：http://localhost:8080
